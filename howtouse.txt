1) git status
2) git add -A (для всех файлов) или git add *.тип_файла
3) git commit -a -m "осознанный коментарий к коммиту"
4) git push

Для того, что бы получить изменения, которые были внесены с другого устройства:
1) git pull

Если происходит слияние(merge) (на одном устройстве было что-то добавлено и при этом на втором устройстве мы эти изменения не внесли(git pull), так ещё и решили внести свои(git push) это и есть слияние)
при попытке git push - ошибка:
делаем git pull;
автоматически начнётся процесс слияния
необходимр вввести описание наших изменений, после этого Ctrl + C два раза, после этого :mq! - Enter
